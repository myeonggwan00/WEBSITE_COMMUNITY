# 📢 Community Web Site

## 📌 프로젝트 소개

**Community Web Site**는 Java와 Spring을 중심으로 한 백엔드 기술을 학습하며, 실습을 통해 개념을 정리하고자 진행한 커뮤니티 웹 애플리케이션입니다.
해당 프로젝트는 **Spring MVC 아키텍처**를 적용하고 SSR 방식으로 구현되었습니다.

## 🛠️ 사용 기술

### 🔙 Back-End
- Java 17
- Spring Boot
- MySQL
- JDBC
- SQL Mapper:JdbcTemplate
- ORM: JPA
- Thymeleaf

### 🔜 Front-End
- HTML5
- CSS3
- JavaScript

## ✨ 주요 기능

### ✅ 사용자 기능
- 게시글 CRUD (작성, 조회, 수정, 삭제)
- 댓글 및 답글 작성 기능
- 파일 업로드 기능
- 회원 가입 / 로그인 / 로그아웃
- 아이디 기억 기능

### 🛡️ 관리자 기능
- 게시글 관리 (작성, 조회, 수정, 삭제)
- 회원 관리 기능
    - 회원 권한 수정 (예: 사용자 → 관리자)
    - 강제 탈퇴 처리
